# Friends Gym

## Description

Friends Gym is a web application designed to provide a professional workout schedule for men and women. The application includes various sections for different muscle groups and an attendance calendar to track workout days.

## Features

- **Responsive Design**: Optimized for all devices.
- **Interactive Attendance Calendar**: Mark and track attendance.
- **Workout Sections**: Includes Front, Back, Legs, and Abs workouts.
- **Modern UI**: Styled with Tailwind CSS and custom animations.

## Technologies Used

- **HTML**: Structure of the application.
- **CSS**: Styling with Tailwind CSS and custom styles.
- **JavaScript**: Logic for interactivity (e.g., attendance tracking, tab switching).

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Mohamed-ashraf-77/LastVersion.git
   ```
2. Navigate to the project directory:
   ```bash
   cd LastVersion
   ```
3. Open `index.html` in your browser to view the application.
